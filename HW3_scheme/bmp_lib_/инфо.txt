Библиотекка для рисования в bmp файле 
Библиотека далеко не полная вы можете сами дополнять ее

для создания изображения нужно создать объект типа Picture

в библиотеку включены 6 функций

void draw() -обязательная функция заносит в файл информацию об изображении
(используется в самом конце когда вы уже решили что нужно нарисовать)

void put_pixel(int x=0,int y=0,int c=6,int r=0,int g=0,int b=0)-рисует пиксель(с -цвет; r,g,b-составляющие для своего цвета)
void put_point(int x=0,int y=0,int c=6,int r=0,int g=0,int b=0) -нарисовать точку 2Х2 пикселей(ствать можно от координат (0,0) до (width-2,height-2) инче будет выход за границы массива )
void line(int x1,int y1,int x2, int y2, int c=6,int r=0,int g=0,int b=0) -нарисовть линию
void circle(int rad,int x0,int y0,int c=6,int r=0,int g=0,int b=0);  -нарисовть круг с центром в точке х0,y0 и радиусом rad
void rectangle(int x0,int y0,int w,int h,int c=6,int r=0,int g=0,int b=0) -нарисовать прямоугольник c центром в точке x0,y0 (w -ширина,h -высота) 


ЦВЕТ:
по умолчанию включено  всего 6 цветов(вы можете сами добавлять нужные)

1- белый
2- красный
3- зеленый
4- желтый
5- голубой
6- черный

для установления произвольного цвета необходимо "с" присвоить значение 7 и 
заполнить составляющие цвета r,g,b
пример : 
object.put_pixel(300,300,7,255,255,255)


Координаты: 
точкой (0,0) считается левый нижний угол рисунка
при создании изображения 600Х600 самая верхняя правая точка будет иметь координаты (599,599)
